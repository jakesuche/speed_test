
# Next.js 

A Test that demostrates typing speed 

This includes Next.js' and  TypeScript.

## How to Start

In your terminal, run the following command:

```bash
yarn add all 
# or
npm install 
```
## Run Application
```bash
yarn dev
# or
npm run dev
```
## Run Jest Tests

```bash
npm test
```
